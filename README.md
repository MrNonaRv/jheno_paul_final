# LGU Payroll Management System

A complete web-based Payroll Management System designed for Local Government Units (LGU).

## Installation

### For Windows Users:
1. Make sure you have **Node.js** installed (Download from [nodejs.org](https://nodejs.org/)).
2. Double-click the `Install-Dependencies.bat` file.
3. Wait for the process to finish.

### For Linux/macOS Users:
1. Open your terminal in this folder.
2. Run `chmod +x install.sh`
3. Run `./install.sh`

## How to Run

### For Windows Users:
1. Double-click `Run-Payroll.bat`.
2. The system will start the server and automatically open your web browser.

### For Linux/macOS Users:
1. Run `npm start` in your terminal.
2. Open your browser and go to `http://localhost:3000`.

## Features
- **Excel Import/Export**: Easily import payroll data from Excel and export reports.
- **Duplicate Deduction Handling**: Automatically handles multiple columns of the same deduction (e.g., FCB Loan 1 & 2).
- **Payslip Printing**: Generate and print professional payslips for individual employees.
- **Data Backup**: Export your entire database to a SQL file for safekeeping.
